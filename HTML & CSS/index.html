What I have learned so far

html tags what they do

  • p tag - the p tag allows you to put text on a page
  • h1 tag - puts a heading. The 1 is interchangeable with 2 or 3. The closer the number is to 1 the bigger the text gets.
  • div tag - separates tags & content inside div tag from surrounding tag styling.
  • span tag - span is similar to to div tag except it goes inside other tags.
  • audio tag - adds audio player to page.
  • a href=”insert link here.com” tag - allows you to put a link on your page
  • img tag - allows you to put image on page.
  • link tag - allows you to link other files to page. Like a css file.
  • object tag - allows you to embed certain types of multimedia.
  • ul tag - this tag creates an unordered list aka bullet point list.
  • ol tag - this tag creates an ordered list aka numbered list.
  • li tag - goes inside ul tag or ol tag. Stands for list item. makes it be a part of the list. Gets a bullet point or number.
